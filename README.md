### This repository contains tools for working with the Celestia blockchain and its elements.

# Celestia - is:

Celestia is a pioneering modular blockchain project designed to address the scalability challenges of traditional blockchains. Launched on October 31, 2023, Celestia differentiates itself by separating data availability from transaction execution, allowing it to handle data availability more efficiently and at scale.

The key innovation behind Celestia is its use of data availability sampling (DAS), which ensures that all data on the blockchain is available and verifiable without requiring every node to store all historical transaction data. This method allows Celestia to offload the heavy lifting of data availability from full nodes, making the network more scalable and decentralized.

Celestia's modular design enables developers to create bespoke execution environments known as rollups. These rollups can be application-specific (sovereign rollups) or general-purpose, and they use Celestia for consensus and data availability. This modularity is akin to a highway system where each road serves a specific purpose, allowing for better handling of increased traffic volumes compared to a single road system.

For developers, Celestia offers various tools and frameworks to easily deploy and manage rollups, making it a promising platform for building scalable blockchain applications.
